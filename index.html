🗺️ Geo Dot-Matrix Label Generator

Generate GeoJSON polygon text labels for embedding in Power BI Shape Maps. Each character is built from rectangular polygons using a 5×7 dot-matrix pixel font.

📋 Paste from Spreadsheet
In Excel or Google Sheets, select your Label, Latitude, and Longitude columns and copy them (Ctrl+C). Paste directly into the box below — a header row is fine, it will be ignored automatically. Rows will appear in the manual editor below so you can review or adjust them.
📝 Labels — review and adjust as needed, or enter manually
All text is automatically converted to UPPERCASE. Enter the bottom-left corner (decimal degrees, WGS84) of each label. Unsupported characters (e.g. accented letters) are silently skipped.

Next Steps — Power BI Shape Map Workflow

1
Download the .geojson file above.
2
Go to mapshaper.org → Import your geography's file (can be GeoJSON or TopoJSON, it doesn't seem to matter) FIRST → also import this GeoJSON → Export both of these layers as TopoJSON.
3
Open geojson.io → import the merged TopoJSON → Meta → Flatten Multi Features. This bakes the letter polygons into the base-map topology so Power BI always renders them. Also, change the file extension from .topojson to just .json (Power BI doesn't recognize the .topojson extension 😔).
4
Load the result into Power BI's Shape Map visual as a custom map.
5
In Power BI: Format pane → Fill colors → Default color to choose your label color. Labels are always visible, independent of the data layer.
💡
To reposition labels, adjust the lat/lng values here and regenerate. You only need to redo the Mapshaper merge when the GeoJSON changes.
